# MBTI 추리 게임

이 프로젝트는 Vite + React로 제작된 MBTI 추리 게임입니다.

## 실행 방법

1. 터미널을 엽니다.
2. `npm install` (이미 설치되어 있다면 생략 가능)
3. `npm run dev`
4. 브라우저에서 `http://localhost:5173` 접속

## 구조
- `src/App.jsx`: 메인 게임 로직
- `src/data.js`: 시나리오 데이터 (질문, 이미지 등)
- `src/index.css`: 스타일링
- `public/images`: 게임에 사용되는 이미지 리소스
